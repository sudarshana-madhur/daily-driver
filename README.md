# Daily Driver

A personal productivity-focused ToDo application designed to keep you focused and motivated. Built using Flutter and deployed as a web application in firebase, Daily Driver fully supports Progressive Web Application (PWA) features, allowing it to be installed and used as an app on mobile devices.

## Overview

Daily Driver is a highly personal task manager. It is not intended to be hosted publicly for general users; rather, it is built specifically to maximize your individual productivity and motivation.

## Features

- **Core Task Management:**
  - Add tasks with ease.
  - Assign due dates.
  - Create recurring tasks.

- **Integrated Kanban Board:**
  - A native Kanban board featuring **ToDo**, **In-Progress**, and **Done** lanes.
  - Unlike traditional task managers where boards might just be for display purposes, moving a task through the lanes in Daily Driver actively updates its status in real-time.
  - Moving a task to the **Done** lane automatically marks it as completed, providing immediate visual feedback and motivation.

- **Status & Analytics Page (Highlight Feature):**
  - **Completion Metrics:** Displays the total number of completed tasks.
  - **Time Tracking:** By logging the "date of in-progress" and "date of completion", the app distinguishes between on-time and late completions.
  - **Productivity Chart:** Combines all your task data to plot a dynamic productivity graph. This visual representation lets you track your growth over time, compare your historical performance, and effectively gamifies your workflow to encourage continuous improvement.

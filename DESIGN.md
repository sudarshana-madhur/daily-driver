# Design System Document

This document outlines the core aesthetic principles and technical specifications of our design system.

## Theme Overview

Our design system is configured with a **dark color mode**, emphasizing a modern and refined user experience.

### Color Palette

The color palette is built around a distinct set of colors to ensure brand consistency and visual hierarchy.

-   **Primary Color:** `#E44332` - A vibrant and commanding color, ideal for call-to-action buttons and key interactive elements.
-   **Secondary Color:** `#504E4B` - A sophisticated supporting color, used for less prominent UI elements and secondary actions.
-   **Tertiary Color:** `#181817` - An additional accent for highlights, badges, or decorative touches, providing depth.
-   **Neutral Color:** `#504E4B` - A versatile base for backgrounds, surfaces, and non-chromatic elements, ensuring readability and visual balance.

### Typography

Our typography system is designed for clarity and impact across all content.

-   **Headline Font:** `inter` - Chosen for its modern aesthetic and strong readability, suitable for all headings.
-   **Body Font:** `inter` - A clean and legible font for all standard text, ensuring comfortable reading experiences.
-   **Label Font:** `inter` - Applied to labels and smaller text elements, maintaining consistency and clarity.

### Shape and Form

The system defines the visual characteristics of UI components.

-   **Roundedness:** `1` (Subtle roundedness) - Provides a touch of softness to UI elements without compromising a sleek appearance.

### Spacing

Whitespace is carefully managed to create visual harmony and improve user experience.

-   **Spacing:** `2` (Normal) - A balanced approach to whitespace, providing adequate breathing room between elements without feeling sparse or cluttered.
